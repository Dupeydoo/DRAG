DRAG - Diversification Rhythm Algorithm Genetics

DRAG is both the name of a rudiment and this platform. DRAG uses an Interactive Genetic Algorithm to diversify a drum track either constructed by the user or from a selection of presets.

When the user has rated a number of generations, a machine learning algorithm runs to learn how the user has rated and automate generations. Single tracks can be downloaded at any time, and can be played easily with keybindings to ease the rating process.
